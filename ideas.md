# Brainstorming de Design — Dr. Santiago Vecina

## Contexto
Site de marca pessoal high ticket para médico, empresário e mentor de Performance Integral.
Público: Empresários brasileiros de alto nível, 35-55 anos, nos EUA.
Referências: Tony Robbins, Ed Mylett, Lewis Howes — mas com identidade própria e brasileira.
Tese Central: "Pessoas alinhadas constroem empresas fortes. Empresas fortes sustentam famílias estruturadas. Famílias estruturadas constroem legado."

**Abordagem 1: "Arquitetura do Poder" — Editorial Escuro de Luxo**

**Design Movement:** Dark Luxury Editorial — inspirado em revistas como Monocle e publicações de moda masculina de alto padrão.

**Core Principles:**
- Contraste extremo: preto profundo + dourado champagne + branco puro
- Tipografia editorial com serifa clássica para títulos (Playfair Display) e sans-serif moderna para corpo (DM Sans)
- Assimetria intencional — layouts que quebram a grade para criar tensão visual
- Fotografia em P&B de alto contraste com sobreposições douradas

**Color Philosophy:** Preto como poder e autoridade. Dourado como conquista e excelência. Branco como clareza e verdade. A paleta comunica: este homem chegou lá, e vai te levar junto.

**Layout Paradigm:** Colunas assimétricas com texto flutuando sobre imagens de fundo escuro. Seções com full-bleed photography intercaladas com blocos de texto denso e elegante.

**Signature Elements:**
- Linhas douradas finas como divisores e ornamentos
- Números romanos para seções (I. QUEM SOU / II. A MISSÃO)
- Citações em destaque com aspas tipográficas grandes em dourado

**Interaction Philosophy:** Hover effects suaves com reveal de dourado. Cursor personalizado. Scroll parallax nas imagens de fundo.

**Animation:** Fade-in de baixo para cima com delay escalonado. Texto que se revela letra por letra nos títulos principais.

**Typography System:** Playfair Display Bold para H1/H2 + DM Sans Regular para corpo. Escala tipográfica dramática (96px → 16px).

**Abordagem 2: "Meridiano" — Minimalismo de Luxo Branco e Dourado**

**Design Movement:** Luxury Minimalism — inspirado em marcas como Hermès, Porsche Design e sites de consultoria de elite.

**Core Principles:**
- Espaço em branco como elemento de luxo — o que NÃO está na tela é tão importante quanto o que está
- Tipografia serifada clássica (Cormorant Garamond) para títulos + Lato para corpo
- Dourado discreto como único acento cromático — nunca gritante, sempre elegante
- Fotografia profissional colorida de alta qualidade como único elemento visual expressivo

**Color Philosophy:** Fundo branco marfim (não branco puro) para suavidade. Texto em grafite escuro. Dourado apenas em detalhes estratégicos (linhas, ícones, CTAs). Transmite: refinamento, confiança, exclusividade.

**Layout Paradigm:** Grid de 12 colunas com muito espaço negativo. Seções amplas e respiradas. Conteúdo centralizado mas com elementos que "sangram" para fora da grade.

**Signature Elements:**
- Linha dourada horizontal fina antes de cada título de seção
- Monograma "SV" em dourado como marca d'água sutil
- Fotografias com moldura sutil ou sem moldura, sempre com muito espaço ao redor

**Interaction Philosophy:** Transições lentas e suaves. Tudo se move como seda. Nada é brusco.

**Animation:** Fade-in muito lento e suave. Scroll-triggered reveals. Nenhuma animação chamativa — apenas elegância discreta.

**Typography System:** Cormorant Garamond Display para títulos grandes + Lato 300/400 para corpo. Muito espaçamento entre letras nos títulos.

**Abordagem 3: "Força e Propósito" — Bold Luxury com Energia de Movimento**

**Design Movement:** Bold Luxury — inspirado em Ed Mylett, Lewis Howes e marcas de performance de alto nível. Poder visível, não escondido.

**Core Principles:**
- Contraste de escala: títulos enormes ao lado de texto pequeno
- Preto profundo + dourado quente + off-white — paleta de poder masculino
- Fotografias de impacto em full-screen como âncoras visuais
- Movimento e energia — o site deve parecer vivo

**Color Philosophy:** Preto como base de poder. Dourado (#C9A84C) como símbolo de conquista e excelência. Off-white (#F5F0E8) para áreas de respiro. A paleta diz: este homem é sério, mas também é ouro.

**Layout Paradigm:** Hero em tela cheia com foto de impacto. Seções alternadas entre fundo escuro e claro. Texto em tamanhos dramáticos. Elementos que "cortam" entre seções com diagonais.

**Signature Elements:**
- Diagonais sutis entre seções (clip-path) para criar dinamismo
- Números grandes em dourado como decoração de fundo (ex: "01", "02")
- Citações da tese central em tipografia gigante como elemento visual

**Interaction Philosophy:** Hover states com glow dourado. Botões com borda dourada que se preenchem ao hover. Sensação de que cada clique é uma decisão de poder.

**Animation:** Framer Motion para entradas dramáticas. Contadores animados para números de impacto. Parallax suave nas imagens de fundo.

**Typography System:** Bebas Neue ou Oswald para títulos de impacto + Nunito Sans para corpo. Contraste máximo entre peso dos títulos e leveza do corpo.
</text>

## Decisão: Abordagem 3 — "Força e Propósito"

Esta é a escolha ideal para o Dr. Santiago Vecina porque:
- Transmite poder e autoridade sem ser arrogante
- A paleta preto + dourado é universalmente reconhecida como high ticket e luxo
- O dinamismo visual reflete a energia de transformação que ele vende
- Alinha com os maiores mentores de referência (Ed Mylett, Lewis Howes) mas com identidade própria
- Funciona perfeitamente para o público-alvo (empresários brasileiros, 35-55 anos)
